# 17.4 Застосування оператора typeof

Незважаючи на всю свою корисність, оператори as і is перевіряють лише сумісність двох типів. Але часто потрібна інформація про сам тип. Для цієї цілі в С# передбачений оператор typeof . Він витягує об'єкт класу System.Туре для заданого типу. За допомогою цього об'єкта можна визначити характеристики конкретного типу даних. Нижче наведена загальна форма оператора typeof:

typeof(тип)

де тип означає отримуваний тип. Інформація, що описує тип, інкапсулюється в повернутому об'єкті класу Туре.

Отримавши об'єкт класу Туре для заданого типу, можна витягнути інформацію про нього, використовуючи різні властивості, поля та методи, визначені в класі Туре. Клас Туре досить обширний і містить багато членів, тому його розгляд доведеться відкласти до наступного розділу, присвяченого рефлексії. Але як введення до цього класу нижче наведена програма, в якій використовуються три його властивості: FullName, IsClass і IsAbstract. Для отримання повної назви типу використовується властивість FullName. Властивість IsClass повертає логічне значення true, якщо тип належить до класу. А властивість IsAbstract повертає логічне значення true, якщо клас є абстрактним.

Лістинг 17.4 - Застосування оператора typeof

using System;
using System.IO;
class UseTypeof {
    static void Main() {
        Type t = typeof(StreamReader);
        Console.WriteLine(t.FullName);
        if(t.IsClass) Console.WriteLine("Належить до класу.");
        if(t.IsAbstract) Console.WriteLine("Є абстрактним класом.");
        else Console.WriteLine("Є конкретним класом.");
    }
}

Ця програма дає наступний результат. System.IO.StreamReader Відноситься до класу. Є конкретним класом. У цій програмі спочатку витягується об'єкт класу Туре, що описує тип StreamReader. Потім виводиться повне ім'я цього типу даних і визначається його належність до класу, а потім — до абстрактного або конкретного класу.
