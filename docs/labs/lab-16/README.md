Порядок виконання роботи

Ознайомитися із теоретичними відомостями пункт 11.9 - 11.15

Додайте нові класи, такі як Truck або ElectricScooter, та реалізуйте специфічні для них властивості та методи.

Перевизначте метод ToString() у кожному з класів для більш зручного виведення інформації про об'єкти.

Контрольні питання

Які переваги та недоліки наслідування у порівнянні з композицією?

Чи можна у C# створити похідний клас від декількох базових класів? Як реалізується схожа функціональність?

Які обмеження накладає модифікатор доступу protected, і як він використовується у наслідуванні?

Що таке інтерфейси, і як вони взаємодіють із наслідуванням у C#?

Яка різниця між інтерфейсом і абстрактним класом у контексті наслідування?

Як у C# реалізувати перетворення типів при роботі з наслідуванням (наприклад, перетворення базового класу до похідного)?

Яке призначення конструктора в базовому класі та як викликати конструктор базового класу з похідного класу?

Чи можна у C# створити похідний клас від фіналізованого класу (sealed class)? Що таке sealed і яка його роль?
