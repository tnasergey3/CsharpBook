Порядок виконання роботи

Ознайомитися з теоретичними відомостями (розділ 2).

Створити програму, яка б виводила на консоль розміри усіх числових типів да їх граничні значення.

Створити програму, яка б обробляла введення строки та числа із консолі, а потім виводила б їх на консоль.

Виконати безкоштовне практичне завдання від Microsoft за посиланням: https://learn.microsoft.com/en-us/dotnet/csharp/tour-of-csharp/tutorials/hello-world. Скриншот кожного виконаного завдання слід додати до звіту.

Результат виконання додати до звіту у вигляді скріншотів. Відповісти на контрольні запитання відповіді додати до звіту.
