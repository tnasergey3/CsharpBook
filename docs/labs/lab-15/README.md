Порядок виконання роботи

Ознайомитися із теоретичними відомостями пункт 11.1 - 11.8

Базовий клас TransportVehicle:

Створіть базовий клас TransportVehicle, який містить:

Властивість string Model — модель транспортного засобу.

Властивість string Brand — марка транспортного засобу.

Властивість int Year — рік випуску транспортного засобу.

Метод void DisplayInfo(), який виводить на екран інформацію про транспортний засіб.

Метод void Start(), який виводить повідомлення про запуск транспортного засобу (наприклад, "Транспортний засіб запущено").

Похідні класи:

Створіть похідні класи від TransportVehicle для різних типів транспортних засобів:

Клас Car:

Додайте властивість int NumberOfDoors – кількість дверей.

Перевизначте метод Start() так, щоб він виводив повідомлення, характерне для запуску автомобіля (наприклад, "Автомобіль запущено").

Клас Motorcycle:

Додайте властивість bool HasSidecar – чи є у мотоцикла боковий причеп.

Перевизначте метод Start() так, щоб він виводив повідомлення, характерне для запуску мотоцикла (наприклад, "Мотоцикл запущено").

Клас Bicycle:

Додайте властивість bool IsElectric — чи є велосипед електричним.

Перевизначте метод Start() так, щоб він виводив повідомлення, характерне для початку руху велосипеда (наприклад, "Велосипед готовий до руху").

Основна програма:

Створіть основну програму, яка виконує наступні дії:

Створює кілька об'єктів кожного з похідних класів (Car, Motorcycle, Bicycle).

Викликає методи DisplayInfo() та Start() для кожного об'єкта.

Використовує поліморфізм для роботи з об'єктами за допомогою базового типу TransportVehicle.
