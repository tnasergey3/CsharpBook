Порядок виконання роботи

Ознайомитися із теоретичними відомостями (підрозділ 6.1).

Виконати практичне завдання за посиланням: https://learn.microsoft.com/en-us/training/modules/csharp-arrays/. Скріншот кожного виконаного завдання слід додати до звіту.

Створити програму яка буде заповнювати масив чисел та виводити їх в консоль. При запуску програми користувач вводить кількість чисел в масиві та самі числа.

Додати до попереднього завдання можливість задати фіксовану кількість повторів виконання програми.

Додати до попереднього завдання можливість розрахунку для обчислення суми усіх введених елементів масиву.

Розробити консольну програму, яка б сортувала введений масив чисел із консолі за зростанням після чого виводила його на екран. Нижче наведений алгоритм сортування методом вибору.

Відповідно до вимог оформити звіт до лабораторної роботи.
