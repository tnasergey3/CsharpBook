Порядок виконання роботи

Ознайомитися із теоретичними відомостями пункт 12

Створіть перелічення PriorityLevel для опису пріоритету завдання

Створіть перелічення TaskStatus для опису статусу виконання завдання

Створіть структуру Task, яка буде представляти завдання

Створіть інтерфейс ITaskManager, який буде містити методи для управління завданнями.

Створіть клас TaskManager, який реалізує інтерфейс ITaskManager. Цей клас буде містити колекцію завдань і реалізовувати методи додавання, видалення, оновлення статусу та відображення завдань.
