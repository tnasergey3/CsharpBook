# Делегати події та лямбда-вирази

[◀️ До змісту лабораторних](README.md)

Порядок виконання роботи

Ознайомитися із теоретичними відомостями пункт 15.

Створіть клас TemperatureSensor з наступними властивостями:

RoomName (string)

CurrentTemperature (double)

У класі TemperatureSensor створіть подію TemperatureChanged, яка викликається при зміні температури.

Створіть делегат TemperatureChangeHandler для обробки змін температури.

Реалізуйте метод SetTemperature(double newTemperature) в класі TemperatureSensor, який оновлює температуру та викликає подію TemperatureChanged.

Створіть клас TemperatureMonitor, який підписується на події від усіх датчиків та виводить повідомлення про зміни температури.

У методі Main:

Створіть кілька екземплярів TemperatureSensor для різних кімнат.

Створіть екземпляр TemperatureMonitor і підпишіть його на події всіх датчиків.

Використовуйте лямбда-вирази для підписки на події та виведення додаткової інформації.

Змініть температуру в різних кімнатах і спостерігайте за виведенням інформації. Додайте функціональність для відстеження критичних змін температури (наприклад, різке підвищення або зниження) за допомогою лямбда-виразів.
