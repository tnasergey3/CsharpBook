Порядок виконання роботи

Ознайомитися із теоретичними відомостями розділу №7.

Завдання 1. Створення класу з полями

Створіть клас Car, який містить наступні публічні поля:

Make (марка автомобіля, тип string)

Model (модель автомобіля, тип string)

Year (рік випуску, тип int)

Напишіть програму, яка створить об'єкт цього класу, задасть значення для його полів і виведе цю інформацію у консоль.

Приклад результату виводу:

Автомобіль: Toyota Corolla, 2015 року випуску.

Завдання 2. Додавання методів у клас

Додайте до класу Car метод GetCarInfo(), який буде повертати рядок з інформацією про автомобіль у форматі:

Марка: <Make>, Модель: <Model>, Рік: <Year>

Напишіть програму, яка створить кілька об'єктів Car, викличе метод GetCarInfo() для кожного з них і виведе результат у консоль.

Завдання 3. Створення конструктора

Розширте клас Car, додавши конструктор, який приймає три параметри (make, model, year) і автоматично ініціалізує відповідні поля об'єкта.

Напишіть програму, яка створить кілька автомобілів за допомогою конструктора та виведе їх інформацію у консоль.

Приклад результату виводу:

Марка: Ford, Модель: Focus, Рік: 2018

Марка: BMW, Модель: X5, Рік: 2020

Завдання 4. Використання властивостей

Модифікуйте клас Car, зробивши поля Make, Model та Year властивостями (використовуйте конструкцію get/set). Додайте перевірку в метод set для властивості Year: рік не може бути меншим за 1886 (перший автомобіль) і більшим за поточний рік. Напишіть програму, яка:

Створює об'єкт Car.

Пробує встановити некоректне значення року.

Виводить інформацію про автомобіль.

Приклад результату виводу:

Некоректний рік: 1700. Будь ласка, введіть рік від 1886 до 2025.

Марка: Honda, Модель: Civic, Рік: 2022
