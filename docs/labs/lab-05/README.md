Порядок виконання роботи

Ознайомитися із теоретичними відомостями (розділ 5).

Виконати практичне завдання за посиланням: https://learn.microsoft.com/en-us/training/modules/csharp-for/. Скріншот кожного виконаного завдання слід додати до звіту.

Виконати практичне завдання за посиланням: https://learn.microsoft.com/en-us/training/modules/csharp-do-while/. Скріншот кожного виконаного завдання слід додати до звіту.

Вивести на екран усі числа від нуля до введеного користувачем числа.

Користувач вводить дві межі діапазону, вивести на екран усі числа з цього діапазону. Передбачити, щоб користувач міг вводити межі діапазону у довільному порядку.

вивести всі парні числа з діапазону.

вивести всі непарні числа з діапазону.

вивести всі числа, кратні семи.

Користувач вводить дві межі діапазону. Порахувати суму всіх чисел діапазону.

Користувач із клавіатури вводить числа. Розрахувати їх суму і вивести на екран, як тільки користувач введе нуль.

Написати програму, яка виводить на екран таблицю множення на k де k – номер варіанта. Наприклад, для 9-го варіанта:

9 x 2 = 18;

9 x 3 = 27.

Відповідно до вимог оформити звіт до лабораторної роботи.

Приклад програми де використовується цикл щоб не завершувати програму після виконання однієї команди.

Лістинг 7.5.1

bool isExit = false;

while (!isExit)

```csharp
Console.WriteLine("Список команд");
Console.WriteLine("1 - вивести число яке ввели");
Console.WriteLine("2 - вивести випадкове число");
Console.WriteLine("3 - вивести строку на екран");
Console.WriteLine("0 - для завершення програми");
int command = int.Parse(Console.ReadLine());
```

if (command == 1)

```csharp
Console.WriteLine("Введіть число");
int arg = int.Parse(Console.ReadLine());
Console.WriteLine($"Ви ввели : {arg}");
```

else if (command == 2)

Random rnd = new Random();

int randomNumber = rnd.Next(1, 100);

```csharp
Console.WriteLine($"Випадкове число : {randomNumber}");
```

else if (command == 3)

```csharp
Console.WriteLine("Hello world");
```

else if (command == 0)

isExit = true;

else

```csharp
Console.WriteLine("Невідома команда використовуйте число від 1 - 3");
```

Контрольні питання

Описати оператор while. Приклад використання.

Описати оператор do ... while. Приклад використання.

Описати оператор for. Приклад використання.

Описати оператор foreach. Приклад використання.

Застосування оператора break. Приклад використання.

Застосування оператора continue. Приклад використання.
