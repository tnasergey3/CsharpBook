Порядок виконання роботи

Ознайомитися із теоретичними відомостями пункт 6.8.

Створити метод, який приймає ведений користувачем рядок та виводить на екран статистику по цьому рядку. Статистика повинна включати загальну кількість символів, кількість літер (і скільки у верхньому та нижньому регістрі), кількість цифр, кількість символів пунктуації та кількість пробілових символів.

Створити програму із наступним функціоналом. Користувач вводить рядок та символ. Потрібно вивести у консоль кількість введеного символу та їх індекси в рядку. Також додати функціонал заміни символів, наприклад, спершу вводимо символ, який користувач бажає замінити та символ на який замінити, а після вивести результати виконання на консоль.

Створити програму із наступним функціоналом. Користувач вводить рядок та символ. У рядку знайти все входження цього символу та перевести його у верхній регістр, а також видалити частину рядка, починаючи з останнього входження цього символу і до кінця рядку.

Створити програму яка зберігає масив імен студентів із функціоналом сортування за алфавітом, видалення по індексу та можливістю змінити ім'я за індексом.

Контрольні питання

Що таке тип string.

Опишіть основні методи для роботи з рядками.

Як створити масив рядків.

Як дізнатися кількість елементів в рядку.
