Порядок виконання роботи

Ознайомитися із теоретичними відомостями (розділ 4).

Створити програму калькулятор із консольним інтерфейсом, спершу ввести тип операції можна використовувати як символи так і числа, а потім два числа після чого вивести результат на консоль.

Створити програму яка буде перевіряти введене ціле число на парність або непарність.

Створити програму яка знаходить найменше та найбільше серед введених трьох чисел та виводить їх в консоль.

Створити програму, яка розв'язує рівняння виду ax + b = c, де a, b та c вводить користувач, після чого програма виводить результат розв'язку або помилку, якщо такого розв'язку знайти не можливо.

Виконати практичне завдання за посиланням: https://learn.microsoft.com/en-us/training/modules/csharp-if-elseif-else/. Скріншот кожного виконаного завдання слід додати до звіту.

Відповідно до вимог оформити звіт до лабораторної роботи.

Приклад програми яка оброблює декілька команд наведений у лістингу 7.4.1.

Лістинг 7.4.1

```csharp
Console.WriteLine("Список команд");
Console.WriteLine("1 - вивести число яке ввели");
Console.WriteLine("2 - вивести випадкове число");
Console.WriteLine("3 - вивести строку на екран");
int command = int.Parse(Console.ReadLine());
```

if (command == 1)

```csharp
Console.WriteLine("Введіть число");
int arg = int.Parse(Console.ReadLine());
Console.WriteLine($"Ви ввели : {arg}");
```

else if (command == 2)

Random rnd = new Random();

int randomNumber = rnd.Next(1, 100);

```csharp
Console.WriteLine($"Випадкове число : {randomNumber}");
```

else if (command == 3)

```csharp
Console.WriteLine("Hello world");
```

else

```csharp
Console.WriteLine("Невідома команда використовуйте число від 1 - 3");
```

Контрольні питання

Для чого використовується оператор if.

Для чого потрібна конструкція if else.

Як працює оператор switch.
