Порядок виконання роботи

Ознайомитися із теоретичними відомостями пункт 14.

Створити програму яка зберігає список введених студентів із консолі та функціонал по збереженню введеного списку в файл якщо ваш номер в групі парний то використовувати текстовий файл якщо не парний то бінарний, також програма повинна підтримувати вивід списку студентів на консоль додавання та видалення можна використовувати як базу виконані завдання з минулих робіт.

Контрольні питання

Для чого потрібен клас Stream

Для чого потрібен клас FileStream

Для чого потрібен клас StreamReader

Для чого потрібен клас StreamWriter

Для чого потрібен клас StringWriter

Для чого потрібен клас StringReader
