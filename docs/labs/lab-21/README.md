Порядок виконання роботи

Ознайомитися із теоретичними відомостями пункт 16.

Створіть два простори імен:

MyApplication.Models

MyApplication.Services

Створення класів:

У просторі імен MyApplication.Models створіть клас User з наступними властивостями:

```csharp
public string Name { get; set; }
public int Age { get; set; }
```

У просторі імен MyApplication.Services створіть клас UserService з наступними методами:

public void AddUser(User user): метод, який приймає об'єкт User і виводить його дані на консоль.

public void DisplayUserInfo(User user): метод, який виводить інформацію про користувача.

Використання класів:

У класі Program (в основному просторі імен) створіть об'єкт User, заповніть його даними, а потім створіть об'єкт UserService і викликати методи AddUser та DisplayUserInfo.
