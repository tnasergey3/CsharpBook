Порядок виконання роботи

Ознайомитися із теоретичними відомостями розділу № 8.

Завдання 1. Організація приватних і публічних полів у класі

Реалізуйте клас BankAccount з наступними полями:

Приватне поле balance (баланс).

Публічне поле AccountNumber (номер рахунку).

Метод Deposit, який додає суму до балансу.

Метод GetBalance, який повертає значення балансу.

Приклад результату виводу:

Баланс рахунку 12345: 1000

Після депозиту 500: 1500

Завдання 2. Передача об'єкта методу для зміни значень

Реалізуйте клас Car з полями Brand (бренд) і Speed (швидкість). Напишіть метод UpgradeCar, який отримує об'єкт Car і збільшує його швидкість на 20%.

Приклад результату виводу:

Автомобіль: BMW, Швидкість: 100

Після покращення: BMW, Швидкість: 120

Завдання 3. Перевантаження методів для розрахунків

Створіть клас MathOperations з перевантаженими методами Calculate:

Один метод приймає два числа і повертає їх суму.

Другий метод приймає три числа і повертає їх добуток.

Приклад результату виводу:

Сума чисел 5 і 10: 15

Добуток чисел 2, 3 і 4: 24

Завдання 4. Використання модифікаторів ref і out

Створіть метод CalculateCircle, який приймає радіус через параметр ref, обчислює площу та довжину кола і повертає ці значення через out.

Приклад результату виводу:

Радіус: 5

Площа: 78.5

Довжина: 31.4

Завдання 5. Статична змінна для підрахунку створених об'єктів

Реалізуйте клас User, який має статичну змінну UserCount, що рахує кількість створених об'єктів, і публічний метод для отримання цього значення.

Приклад коду:

Counter c1 = new Counter();

Counter c2 = new Counter();

Counter c3 = new Counter();

Counter.ShowCount();

Приклад результату виводу:

Кількість користувачів: 3

Контрольні питання

Дати визначення поняттю доступ до членів класу.

Дати визначення поняттю модифікатори доступу.

Дати визначення поняттю необов'язковий аргумент методу.

Дати визначення поняттю перевантаженню методів класу.

Для чого використовуються ключове слово static.
